# Stremio Addons and Extensions

This directory contains Stremio-focused projects by Chimera Gaming.

It currently hosts [Chimera Folders](./Chimera%20Folders), a custom extension built to improve the [Stremio](https://www.stremio.com/) web experience with better organization tools and a cleaner library workflow.

---

## What Is Included

- [Chimera Folders](./Chimera%20Folders)

The Chimera Folders project includes source code, manifest files, icons, a changelog, and a dedicated README with features and installation details.

---

## Current Project

### Chimera Folders

Organize your Stremio library into custom folders with a collapsible sidebar, quick navigation, and integrated project links.

More Stremio projects may be added here over time.

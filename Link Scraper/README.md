# Link Scraper

Link Scraper is a Chromium-based extension that opens a floating panel on the current page and collects visible `http` and `https` links.

It is built for quick filtering, lightweight cleanup, optional descriptions, and `.meta4` export for download manager workflows.

---

## Features

- Scrape links directly from the current page
- Filter results with include and exclude terms
- Use Edit mode to remove links from the current result set
- Use Descriptions mode to attach notes to links
- Save filtered links as a `.meta4` file
- Work from a draggable, resizable overlay panel

---

## Manual Installation

1. Clone or download this repository.
2. Open `chrome://extensions/`.
3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the `Link Scraper` folder.

---

## Basic Usage

1. Open the page you want to inspect.
2. Click the extension icon to open the overlay.
3. Click **Scrape links**.
4. Use **Filter In** and **Filter Out** if needed.
5. Switch to **Edit** or **Descriptions** mode when you want to refine the list.
6. Save the filtered result as `.meta4` when ready.

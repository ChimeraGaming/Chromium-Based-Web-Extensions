# Changelog

## v2.1
- Added **Filter In** and **Filter Out** boxes under Scrape links.
- Supports multiple terms (comma or semicolon separated).
- Pressing **Enter** in either box applies filters.
- Filters always run on the full scraped list.
- Filter In keeps matches, Filter Out removes matches.
- Status line shows “Filtered X > Y links.” or “Filter cleared.”
- Copy all now copies only filtered URLs, one per line.

## v2.0
- Added an inline filter bar.
- Enter key applies filtering by URL or description.
- Status now reports number of matches.
- Copy all copies only filtered URLs (no descriptions).

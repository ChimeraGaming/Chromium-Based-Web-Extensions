Roadmap

Last updated: March 26, 2026 (v2.0)

New in v2.0

  • Major UI cleanup across panel and settings.
  • Panel is draggable and auto closes after successful scan results open.
  • Main upload flow is now one button with ZIP and Folder picker.
  • Scan mode flow was simplified into clear mode names.
  • Added Fabric mod update mode for direct JAR metadata update.
  • Added info bubbles for scan mode and target version.
  • Added source count indicator in panel and settings.
  • Added filter-aware Download All alongside `.meta4` export.
  • Compare mode upload layout was cleaned up and organized side by side.

Fixed in v2.0

  • Compare mode now keeps valid rows and avoids over-filtering matched identities.
  • Compare mode now checks matched target file data before showing a mod as upgradable or downgradable.
  • Compare old and new matching now uses project keys and target file stems for better accuracy when local mod versions do not include game version text.
  • Fabric update mode now resolves the correct minimum available Fabric Loader version for the selected target game version.
  • Fabric update mode now edits only minecraft and fabricloader dependencies.
  • Close button sizing and corner placement no longer overlap long panel titles.
  • Version label cleanup completed for `26.1` references.

Feedback Focus

  • Compare results that look too short for the selected packs.
  • Fabric update mode edge cases where dependency strings vary from standard patterns.
  • Export and Download All cases where filtered rows do not match expected links.
